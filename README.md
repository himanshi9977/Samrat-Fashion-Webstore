#  Samrat Fashion

![Samrat Fashion Banner](https://via.placeholder.com/1000x300?text=Samrat+Fashion+-+The+Elegance+of+Ethnic+Wear)

**Samrat Fashion** is a modern e-commerce platform dedicated to high-end ethnic wear. It blends traditional Indian aesthetics with a seamless digital shopping experience, featuring curated collections of Sarees, Lehengas, and Menswear.

---

##  Features

*   **Curated Collections:** Specialized sections for Weddings, Festivals, and Daily Wear.
*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
*   **Dynamic Filtering:** Search by fabric, color, price range, and occasion.
*   **Secure Checkout:** Integrated payment gateway support.

##  Tech Stack

*   **Frontend:** HTML5, CSS3 (SCSS), JavaScript (ES6+)
*   **Frameworks:** Booststrap5
*   **Deployment:** Hosted via Netlify.
